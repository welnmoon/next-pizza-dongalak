Next Pizza — e-commerce приложение для онлайн-заказа пиццы,
реализованное на Next.js с использованием App Router.

Проект демонстрирует полный цикл e-commerce:
каталог, корзину, оформление заказа, онлайн-оплату,
аутентификацию пользователей и административную панель.

Что здесь есть
- Каталог и карточка товара с выбором параметров (размер, тип теста, ингредиенты)
- Корзина и оформление заказа с онлайн‑оплатой
- Регистрация/вход и личный профиль с историей заказов
- Базовая админ‑панель для управления товарами, категориями и ингредиентами

Проект создан как демонстрация базового e‑commerce функционала в современном фронтенд‑подходе.
